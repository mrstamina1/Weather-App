# Weather App - Real-Time

A simple **Python Tkinter Weather App** that shows the current weather for selected cities using the **Open-Meteo API**.  
No API key is required, and it fetches real-time temperature, weather conditions, and wind speed.

## Features
- Real-time weather from Open-Meteo
- Easy-to-use Tkinter interface
- Clear display of temperature, condition, and wind

## Supported Cities
- Tehran
- Mashhad
- Isfahan
- Shiraz
- Tabriz
- Karaj
